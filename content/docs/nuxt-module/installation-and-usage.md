Nuxt module can be installed only via CLI like NPM or Yarn.


## Installation

CLI
```bash
yarn add -D @stylify/stylify

npm i -D @stylify/stylify
```

## Usage

Add a buildModule into the nuxt.config.js
```js
buildModules: [
	'@stylify/nuxt-module'
]
```
